AudioRecordingPresetOverride

enum class AudioRecordingPresetOverride

AudioRecordingPresetOverride describes the audio recording preset in which the audio client should operate during a meeting session. The values below (except None) directly map to the values defined in: https://github.com/ricardoquesada/android-ndk/blob/master/usr/include/SLES/OpenSLES_AndroidConfiguration.h

Enum Values

None

No preset override. We will use default preset depending on the mode of operation

Generic

Equivalent of SL_ANDROID_RECORDING_PRESET_GENERIC in openSLES

Camcorder

Equivalent of SL_ANDROID_RECORDING_PRESET_CAMCORDER in openSLES

VoiceRecognition

Equivalent of SL_ANDROID_RECORDING_PRESET_VOICE_RECOGNITION in openSLES

VoiceCommunication

Equivalent of SL_ANDROID_RECORDING_PRESET_VOICE_COMMUNICATION in openSLES
